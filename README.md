Eco-Sounds: Plataforma de Difusión Cultural y Conciencia Social

 Integrante:
* Andy

 Propósito del Proyecto:
Analizar el impacto de las nuevas tecnologías en la sociedad mediante una herramienta digital que promueve el contenido educativo (pódcasts de salud mental e inteligencia artificial) y apoya la difusión de artistas musicales locales emergentes, fomentando un uso responsable y equilibrado de las plataformas de streaming.

Funcionalidades del Sistema:
1. **Catálogo de Impacto:** Despliegue de listas dinámicas con contenido seleccionado.
2. **Control de Bienestar Digital:** Validación mediante estructuras lógicas para limitar la reproducción excesiva y promover la salud digital.
3. **Módulo de Estadísticas:** Funciones optimizadas bajo el paradigma funcional para calcular minutos de uso y registros históricos.
4. **Filtro Avanzado:** Uso de funciones puras para clasificar y separar pódcasts educativos de forma eficiente.
